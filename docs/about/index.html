Civkit About

CivKit is an open-source project which maintains and releases CivKit node client and associated softwares. The project is carried out with utmost transparency. Each development decision is open to public discussion. Anyone is free to propose code changes and to test, review and comment open pull request. The CivKit project is committed to the highest degree of decentralization both in network design and code development.

The technical philosophy of the project is driven to protect the security, safety and privacy of the CivKit users.
